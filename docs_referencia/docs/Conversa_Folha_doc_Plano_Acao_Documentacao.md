Ambiente validado: figmm

## 1. Objetivo

Definir ações práticas para manter a documentação coerente com o sistema e com a governança FACIN_IA.

## 2. Ações Imediatas

| Prioridade | Ação | Resultado esperado |
| --- | --- | --- |
| Imediata | Executar o gerador após qualquer alteração relevante no legado | docs sincronizados |
| Imediata | Revisar a avaliação de maturidade a cada novo ciclo de entrega | leitura executiva atualizada |
| Alta | Atualizar o registro de erros quando novos desvios forem encontrados | trilha de auditoria preservada |
| Alta | Revisar o agente FACIN_IA quando o escopo documental mudar | comportamento do agente alinhado ao projeto |
| Média | Submeter o pacote documental a validação de negócio e privacidade | maior aderência institucional |

## 3. Evidências Esperadas

1. arquivos gerados em md, html e pdf;
2. histórico de achados na pasta errors;
3. avaliação de maturidade com data e contexto;
4. agente FACIN_IA coerente com o projeto vigente.
